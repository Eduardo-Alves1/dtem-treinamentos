# DTEM Treinamentos - Landing Page

Primeira versão da landing page estática da formação inicial para carreira em tecnologia.

## Estrutura

- `index.html`: conteúdo da página
- `styles.css`: identidade visual e layout responsivo
- `script.js`: comportamento simples do formulário demonstrativo

## Publicação no GitHub Pages

1. Suba a pasta `landpage` para o repositório no GitHub.
2. Acesse `Settings` > `Pages`.
3. Em `Build and deployment`, selecione `Deploy from a branch`.
4. Escolha a branch principal e a pasta raiz do repositório.
5. Salve e acesse a página pelo caminho `/landpage/` dentro da URL gerada pelo GitHub Pages.

Alternativas:

- mover os arquivos de `landpage` para uma pasta `docs`;
- mover os arquivos para a raiz de uma branch dedicada, como `gh-pages`;
- configurar um workflow do GitHub Actions para publicar apenas a pasta `landpage`.
