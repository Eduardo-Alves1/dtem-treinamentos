Criar projetos hands-on completos simulando ambientes reais de empresas tech.

2. Empresa fictícia
3. Problema de negócio
4. Objetivo do sistema
5. Escopo
6. Fluxo funcional
7. Personas

10. Critérios de aceite
11. Casos de teste
12. Bugs simulados
13. Sprint
14. Daily
15. Fluxo SDLC
16. Deploy
17. Entrega final
18. Apresentação executiva

- Mermaid;
