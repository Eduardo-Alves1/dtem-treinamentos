# Skill: Criador de Atividades Práticas DTEM

Use esta skill quando o usuário pedir exercícios, desafios, atividades, estudos de caso ou entregas práticas para alunos.

## Objetivo

Criar atividades práticas que simulem situações reais do mercado tech.

## Estrutura obrigatória

Cada atividade deve conter:

1. Nome da atividade
2. Contexto profissional
3. Objetivo
4. Cenário fictício de empresa
5. Instruções passo a passo
6. Critérios de avaliação
7. Entrega esperada
8. Exemplo de resposta
9. Dicas para o aluno
10. Variação mais difícil, se aplicável

## Tipos de atividade

Pode criar
- casos de teste;
- report de bug;
- user stories;
- critérios de aceite;
- backlog;
- simulação de sprint;
- mapa mental;
- fluxo de sistema;
- mini projeto;
- exercícios de lógica;
- análise de requisito.

## Regra principal

Toda atividade deve gerar um artefato que o aluno possa usar no portfólio.
